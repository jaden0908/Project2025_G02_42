1. Installation Manual
------------------------------------------
1. Download the project from GitHub.
2. Extract the ZIP file into htdocs (for XAMPP).
3. Open phpMyAdmin and create a new database named 'waterland'.
4. Import the 'waterland.sql' file.
5. Start Apache and MySQL in XAMPP Control Panel.
6. Open your browser and go to:
   http://localhost/universal studios/
